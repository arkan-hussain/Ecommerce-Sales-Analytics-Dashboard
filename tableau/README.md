# Tableau Files

  dashboard sheets and packaged CSV data.
- `Ecommerce_Sales_Analytics_Dashboard.twb`: unpackaged workbook XML.
- `Data/sales_orders_tableau.csv`: Tableau-friendly cleaned dataset.
- `Tableau_Dashboard_Build_Guide.md`: exact field, sheet, filter, and layout
  instructions for final polish inside Tableau Desktop.

Open the `.twbx` file in Tableau Desktop 2026.2 or newer.
